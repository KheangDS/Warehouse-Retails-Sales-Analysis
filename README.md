# Warehouse & Retail Sales Analysis Dashboard (Power BI)

## Overview

This project presents an interactive Power BI dashboard developed to analyze warehouse and retail sales data from Montgomery County, Maryland. The dashboard provides insights into sales performance, supplier contributions, product trends, inventory movement, and seasonal sales patterns.

The goal of this project is to transform raw transactional sales data into meaningful business intelligence that supports data-driven decision-making, sales monitoring, and trend analysis.

---



## Dataset Information

The analysis uses the **Warehouse and Retail Sales** dataset published by Montgomery County, Maryland Open Data Portal.

### Dataset Source

* Montgomery County Open Data Portal
* Dataset: Warehouse and Retail Sales
* Update Frequency: Monthly
* Data Category: Community / Recreation
* Public Domain Dataset

Dataset Link:
https://data.montgomerycountymd.gov/Community-Recreation/Warehouse-and-Retail-Sales/v76h-r7br

### Dataset Description

The dataset contains monthly records of warehouse and retail product sales, including product information, supplier information, and inventory movement metrics.

### Features

| Column           | Description                                |
| ---------------- | ------------------------------------------ |
| Year             | Calendar Year                              |
| Month            | Month Number                               |
| Supplier         | Product Supplier                           |
| Item Code        | Unique Product Identifier                  |
| Item Description | Product Name                               |
| Item Type        | Product Category                           |
| Retail Sales     | Cases sold through retail dispensaries     |
| Retail Transfers | Cases transferred between retail locations |
| Warehouse Sales  | Cases sold through warehouse operations    |

The dataset contains more than 300,000 records and is updated monthly by Montgomery County's Department of Liquor Control.

## Project Objectives

This dashboard was designed to:

* Analyze overall retail and warehouse sales performance.
* Identify top-performing suppliers and products.
* Monitor sales trends over time.
* Compare warehouse sales against retail sales.
* Detect seasonal patterns and demand fluctuations.
* Support inventory and supply chain decision-making.
* Provide an interactive business intelligence solution using Power BI.

---

## Data Cleaning & Transformation

Data preprocessing was performed using Power Query in Power BI.

### Cleaning Steps

* Removed null and missing values.
* Corrected data types.
* Converted month numbers into month names.
* Standardized supplier names.
* Built calculated measures using DAX.

## Key Performance Indicators (KPIs)

* Total Retail Sales
* Total Warehouse Sales
* Total Retail Transfers

* Top Product

* Year-over-Year Sales Change
